# Hello Deutsch Website

This is the frontend code for the Hello Deutsch language learning platform.

## Important Note

This code is a snapshot of a project built on the base44 platform. It requires base44's backend services to be fully functional.

This code **cannot run standalone** without the base44 infrastructure and is intended for backup and version control purposes.

## Live Website

The live, functional version of this application is hosted by base44.

## Technology Stack

- React
- Tailwind CSS
- shadcn/ui components
- Lucide React for icons
